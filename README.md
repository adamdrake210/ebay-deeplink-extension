# WebDev eBay App Native Link Generator

## This extension will create eBay App native links for any valid eBay link.

## How to install

1. Clone repo
2. Run "npm build"
3. Go to [Chrome extensions](chrome://extensions/)
4. Click "Load unpacked"
5. Upload the "dist" folder
6. Ready to go !
